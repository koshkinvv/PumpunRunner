О Telegram-боте для бегунов

Описание бота

Бот для бегунов - это мощный инструмент для создания персонализированных планов тренировок и отслеживания прогресса. Бот помогает бегунам готовиться к соревнованиям, поддерживать мотивацию и достигать результатов.

Ключевые возможности:
  • Создание индивидуальных планов тренировок на основе профиля бегуна
  • Автоматическое распознавание скриншотов из популярных трекеров (Strava, Nike Run, Garmin)
  • Отслеживание прогресса и еженедельного объема бега
  • Интеллектуальная корректировка плана при значительных отклонениях от плана
  • Статистика и аналитика тренировок
Технологии
Backend:
  • Python
  • python-telegram-bot для работы с Telegram API
  • OpenAI API для создания персонализированных планов и анализа скриншотов
  • SQLAlchemy ORM
  • PostgreSQL для хранения данных
Frontend:
  • Flask для веб-интерфейса администрирования
  • Bootstrap CSS для стилизации
Инфраструктура:
  • Многоуровневая система мониторинга и стабильности
  • Автоматический перезапуск при сбоях
  • Ротация логов и контроль использования ресурсов
  • Поддержка Reserved VM для непрерывной работы
Команды бота
Команда Описание
/start Начать работу с ботом и создать профиль бегуна
/help Показать справку по командам бота
/plan Создать новый план тренировок
/profile Показать ваш текущий профиль бегуна
/trainings Показать ваш текущий план тренировок
/pending Показать только невыполненные тренировки
/stats Показать статистику ваших тренировок

Кроме того, вы можете отправить боту скриншот из приложения для бега (Strava, Nike Run, Garmin), и он автоматически распознает данные тренировки и отметит соответствующий день в вашем плане как выполненный.
